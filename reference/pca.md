# Perform PCA

Perform PCA. Outputs a list with dataframe and samples colouring info
ready for plotting.

## Usage

``` r
pca(data, targets, title = "", report_dir, suffix = "")
```

## Arguments

- data:

  Input data.

- targets:

  Target group.

- title:

  Title.

- report_dir:

  Report directory.

- suffix:

  Suffix for output file.

## Value

Outputs a list with data frame and samples colouring info ready for
plotting.
