# Pre-Process Arriba or DRAGEN Fusions

Pre-Process Arriba or DRAGEN Fusions

## Usage

``` r
fusions_preprocess(d, known_translocations, genes_cancer)
```

## Arguments

- d:

  Parsed tibble with Arriba or DRAGEN fusions.

- known_translocations:

  Tibble with known translocations.

- genes_cancer:

  Character vector of cancer genes.

## Value

Tibble with post-processed fusion calls.
