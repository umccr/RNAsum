# Get PCGR genes with hits

Get PCGR genes with hits

## Usage

``` r
pcgr_expr(pcgr = NULL, expr_data.z, expr_data.perc)
```

## Arguments

- pcgr:

  Path to parsed PCGR tiered TSV tibble if available.

- expr_data.z:

  Tibble with gene symbol and z-score expr values.

- expr_data.perc:

  Tibble with gene symbol and percentile expr values.

## Value

List.
