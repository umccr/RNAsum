# Process Arriba Fusions

Process Arriba Fusions

## Usage

``` r
arriba_process(arriba.fusions, known_translocations, genes_cancer)
```

## Arguments

- arriba.fusions:

  Parsed tibble with Arriba fusions.

- known_translocations:

  Tibble with known translocations.

- genes_cancer:

  Character vector of cancer genes.

## Value

Tibble with post-processed Arriba fusion calls.
