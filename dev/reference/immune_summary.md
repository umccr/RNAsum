# Get Immune Gene Summary

Get Immune Gene Summary

## Usage

``` r
immune_summary(tbl_imarkers, tbl_igram = NULL, igram_param = TRUE)
```

## Arguments

- tbl_imarkers:

  Tibble with immune response marker genes.

- tbl_igram:

  Tibble with immunogram genes.

- igram_param:

  Immunogram inclusion TRUE/FALSE parameter.

## Value

Character vector of genes.
